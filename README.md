# Dynime ERP

> A modern, full-stack Enterprise Resource Planning system for comprehensive business management and automation.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.0+-blue.svg)](https://www.typescriptlang.org/)
[![Laravel](https://img.shields.io/badge/Laravel-12-red.svg)](https://laravel.com)
[![React](https://img.shields.io/badge/React-18-blue.svg)](https://react.dev)

Dynime ERP is an integrated business management platform combining a powerful Laravel backend with a modern React frontend. It streamlines operations across invoicing, inventory management, employee payroll, CRM, and more—with real-time synchronization, multi-channel payment processing, and enterprise-grade security.

## ✨ Key Features

### 📊 Business Operations
- **Invoicing & Billing** — Generate, track, and manage invoices with automated payments
- **Inventory Management** — Real-time stock tracking, warehouse operations, and supply chain
- **Employee Management** — HR records, attendance, performance tracking
- **Payroll Processing** — Salary calculations, tax management, payment disbursement
- **CRM Integration** — Customer relationship management and sales pipeline

### 🔄 Integration & Communication
- **Multi-Payment Gateway Support** — Stripe, PayPal, Razorpay, Aamarpay
- **Real-time Notifications** — Pusher WebSocket integration for live updates
- **Email & SMS** — Twilio integration for SMS notifications, mailing system
- **Google Services** — Calendar sync, contact integration, Google Drive
- **Social Authentication** — OAuth2 via Laravel Socialite

### 📅 Advanced Features
- **Event Scheduling** — FullCalendar with drag-and-drop support
- **Rich Content Creation** — TipTap WYSIWYG editor for documents
- **Diagram & Flowcharts** — Syncfusion diagrams for process visualization
- **Data Export** — Excel spreadsheets, PDF generation (jsPDF)
- **Barcode & QR Codes** — JsBarcode and QR code generation
- **Calendar Management** — Spatie Google Calendar integration
- **Media Management** — Spatie media library for file handling

### 🔐 Security & Access
- **Role-Based Access Control** — Spatie permissions and roles
- **API Authentication** — Laravel Sanctum for secure API access
- **Two-Factor Authentication** — Google Authenticator support
- **reCAPTCHA Protection** — Form spam prevention
- **Audit Logging** — Activity tracking and compliance

### 🌍 Internationalization
- **Multi-Language Support** — i18next for frontend localization
- **Multiple Locales** — Timezone and locale configuration
- **Translation Management** — Extract and manage translations

## 🏗️ Tech Stack

### Backend
- **PHP 8.2+** with **Laravel 12** framework
- **MySQL 8.0+** database
- **RESTful API** architecture
- **Eloquent ORM** for database operations

### Frontend
- **TypeScript 5.0+** for type safety
- **React 18** with Hooks
- **Inertia.js** for server-side routing
- **Tailwind CSS 3** with Radix UI components
- **Vite 5** for blazing-fast builds

### Key Dependencies
| Layer | Technology |
|-------|------------|
| **Real-time** | Pusher, Laravel Echo |
| **Payments** | Stripe, PayPal, Razorpay |
| **UI Components** | Radix UI, Headless UI |
| **Rich Editor** | TipTap |
| **Calendar** | FullCalendar |
| **Charts** | Recharts |
| **Spreadsheet** | x-data-spreadsheet |
| **Documents** | jsPDF, html2pdf.js |
